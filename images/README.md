# 员工照片说明

## 目录结构
- avatars/ - 存放员工照片的目录

## 照片要求
1. 文件格式：支持 jpg、png、webp 格式
2. 文件命名：使用员工ID作为文件名（例如：1.jpg、2.jpg）
3. 照片尺寸：建议使用正方形照片，推荐尺寸 400x400 像素或更大
4. 照片内容：建议使用正面照，背景清晰

## 使用说明
1. 将员工照片放入 avatars/ 目录
2. 确保照片文件名与 employees.json 中的 ID 对应
3. 所有照片尺寸保持一致，以确保显示效果的统一性

注意：请确保拥有照片使用权，并事先获得员工同意。
